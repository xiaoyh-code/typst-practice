## 什麼是 Typst？ / What is Typst?

<div class="bilingual-block">
<div class="zh-block">
<span class="lang-label">中文</span>

**Typst** 是一種基於標記（markup）的新型排版系統，結合了輕量級標記語法與強大的指令碼功能。與 LaTeX 相比，Typst 更易學易用，同時保持專業排版品質。

- 內建常見格式的標記語法
- 靈活的函數系統
- 緊密集成的指令碼功能
- 數學排版、參考文獻管理
- **極快的增量編譯速度**
- **友好的錯誤訊息**

</div>
<div class="en-block">
<span class="lang-label">English</span>

**Typst** is a new markup-based typesetting system that combines lightweight markup syntax with powerful scripting capabilities. Compared to LaTeX, Typst is easier to learn and use while maintaining professional typesetting quality.

- Built-in markup for common formatting
- Flexible function system
- Tightly integrated scripting
- Math typesetting, bibliography management
- **Extremely fast incremental compilation**
- **Friendly error messages**

</div>
</div>
