# 互動沙盒 / Interactive Sandbox

<div class="bilingual-block">
<div class="zh-block">
<span class="lang-label">中文</span>

在左側編輯器輸入 Typst 代碼，右側會**即時顯示編譯結果**（首次載入引擎需 5–15 秒）。編譯失敗時，錯誤面板會顯示行號與診斷訊息。下方還有**練習抽認卡**，翻面看答案、一鍵載入編輯器練習！

</div>
<div class="en-block">
<span class="lang-label">English</span>

Type Typst code in the left editor and the right panel shows the **live compiled preview** (the engine takes 5–15 seconds to load the first time). If compilation fails, the error panel shows line numbers and diagnostics. Below you'll find **practice flash cards** — flip for answers and load them into the editor with one click!

</div>
</div>

<div class="tip-box">

此沙盒使用 <a href="https://github.com/Myriad-Dreamin/typst.ts" target="_blank">typst.ts</a> 在瀏覽器內以 WASM 編譯，**不需要安裝任何軟體**。若需完整功能（套件、自訂字型、雲端專案），可使用 <a href="https://typst.app/play/" target="_blank">Typst 官方線上編輯器</a>。

This sandbox compiles Typst in your browser via WASM using <a href="https://github.com/Myriad-Dreamin/typst.ts" target="_blank">typst.ts</a> — **no installation required**. For full features (packages, custom fonts, cloud projects), use the <a href="https://typst.app/play/" target="_blank">official Typst online editor</a>.

</div>
