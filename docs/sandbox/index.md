# 互動沙盒 / Interactive Sandbox

<div class="bilingual-block">
<div class="zh-block">
<span class="lang-label">中文</span>

在左側編輯器輸入 Typst 代碼，右側會**即時顯示編譯結果**（首次載入引擎需 5–15 秒）。編譯失敗時，錯誤面板會顯示行號與診斷訊息。下方還有**練習抽認卡**，翻面看答案、一鍵載入編輯器練習！

</div>
<div class="en-block">
<span class="lang-label">English</span>

Type Typst code in the left editor and the right panel shows the **live compiled preview** (the engine takes 5–15 seconds to load the first time). If compilation fails, the error panel shows line numbers and diagnostics. Below you'll find **practice flash cards** — flip for answers and load them into the editor with one click!

</div>
